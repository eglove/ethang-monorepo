# Initial Concept

This monorepo contains a collection of web applications, libraries, and tools used for personal and client projects.

## Product Vision

The overarching goal of this monorepo is to provide a single, maintainable repository for both personal and client projects while streamlining the sharing of core libraries (such as hooks, fetch wrappers, and toolbelts) across diverse applications.

## Target Audience

The applications and packages within this monorepo serve a diverse set of users:
- **Primary Users:** Myself (for personal productivity and portfolio) and specific clients.
- **Developers:** Other developers utilizing the open-source shared packages (`@ethang/hooks`, `@ethang/store`, etc.).
- **Community:** The general public and specific community members (e.g., Sterett Creek residents).

## Future Expansion

Moving forward, the monorepo will expand by:
- Growing the open-source library ecosystem to support a wider developer community.
- Adding new applications that address emerging needs, which may not necessarily be tied to the existing technology stack.
