# Remove Flowbite & Introduce Colocated Client Scripts

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove Flowbite from ethang-hono, replace it with custom styles and a colocated `.client.ts` build convention that server-registers per-page scripts into `<head>` exactly once.

**Architecture:** Components call `registerScript(...ids)` during SSR to add script IDs to a request-scoped `Set<string>` on `globalStore`. `MainLayout` reads the set and emits `<script type="module">` tags in `<head>`. A build step globs `src/**/*.client.ts`, emits their path-derived IDs into `src/generated/script-ids.ts` as a `ScriptId` union type, then bundles them all to `public/scripts/` mirroring the source tree (stripping `src/`).

**Tech Stack:** Hono/JSX (server rendering), Vitest (tests), tsup (client bundling), fast-glob, Tailwind CSS v4

## Task 4: Update build.ts to generate ScriptId and use new glob

`build.ts` must: (1) glob all `src/**/*.client.ts` files, (2) derive their IDs by stripping `src/` and `.client.ts`, (3) write `src/generated/script-ids.ts` with a `ScriptId` union, (4) pass the files to tsup with path-preserving entry names.

- [ ] **Step 2: Update build.ts**

Replace the entire content of `apps/ethang-hono/build.ts`:

```ts
import { writeFileSync } from "node:fs";

import { globSync } from "fast-glob";
import map from "lodash/map.js";
import reduce from "lodash/reduce.js";
import replace from "lodash/replace.js";
import { build as tsup } from "tsup";

const CLIENT_GLOB = "src/**/*.client.ts";

const entries = globSync(CLIENT_GLOB);

// Derive the ScriptId for each file: strip "src/" prefix and ".client.ts" suffix.
const scriptIds = map(entries, (entry) =>
  replace(replace(entry, "src/", ""), ".client.ts", ""),
);

// Generate src/generated/script-ids.ts
const unionBody =
  scriptIds.length === 0
    ? "never"
    : map(scriptIds, (id) => `  | "${id}"`).join("\n");

const generated = `// This file is auto-generated by build.ts — do not edit manually.
// Run \`bun build.ts\` (or the deploy script) to regenerate.

export type ScriptId =
${unionBody};
`;

writeFileSync("src/generated/script-ids.ts", generated, "utf8");

// Build entry map: preserve directory structure relative to src/
// e.g. src/components/navigation/navigation.client.ts
//   → public/scripts/components/navigation/navigation.client.js
const entryMap = reduce(
  entries,
  (accumulator, entry) => {
    const key = replace(replace(entry, "src/", ""), ".ts", "");
    accumulator[key] = entry;
    return accumulator;
  },
  {} as Record<string, string>,
);

await tsup({
  bundle: true,
  clean: false,
  entry: entryMap,
  format: ["esm"],
  injectStyle: true,
  minify: true,
  noExternal: [/./u],
  outDir: "public/scripts",
  platform: "browser",
  sourcemap: false,
  target: "esnext",
});
```

        try {
          var url = new URL(a.href);
          if (url.origin !== location.origin) return;
          var key = url.pathname + url.search;
          if (seen[key]) return;
          seen[key] = true;
          urls.push(url.origin + key);
        } catch {}
      });
      if (urls.length > 0 && registration.active) {

      }
    });
  });
  navigator.serviceWorker.addEventListener('message', function (event) {
    if (event.data && event.data.type === 'CONTENT_UPDATED') {
