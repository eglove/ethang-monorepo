Elicitor

Interactive

Opens a new terminal for a freeform interview. One question at a time until all design dimensions are resolved.

In Seed prompt
Out docs/<feature>/elicitor.md

Out bdd.feature

Loop

Expert panel debates the BDD scenarios against the original briefing. On consensus, the recommendation is applied by the writer.
